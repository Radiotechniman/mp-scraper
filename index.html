    Marktplaats Scraper

    Want to know the average price on Marktplaats for a product you want to sell? I would!

    Installation

    git clone git://github.com/manuelvanrijn/mp-scraper.git
cd mp-scraper
npm install

    Example

    Let's say I want to sell my old iPhone 4. On Marktplaats I'll setup some filters (above 100 EUR, not the 3gs etc.). After you finshed filtering, copy the url.

    In this example we'll us the iPhone 4 example url

    $ bin/mp-scraper
